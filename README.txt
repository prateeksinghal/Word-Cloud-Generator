This Repository has "The Word Cloud Generator" for a huge data set containing the Medical Text/Documents.
Word-clouds recently emerged on the web as a solution for quickly summarizing text by maximizing the display of most
relevant terms about a specific topic in the minimum amount of space.

Steps of Using Word Cloud Generator:

1. Open cmd(For Windows)/Terminal(For Linux)

2. Go to Directory Genes2World

3. Execute the following commands.
   $ javac cloud.java
   $ java cloud

4. Enter the path of "Input" folder located inside docs-1 folder.

After these steps you can see the Word Cloud of the Top-100 most frequent words in the Data-set.

The entire implementation of this project is done in Object Oreinted Programming Language - JAVA.
